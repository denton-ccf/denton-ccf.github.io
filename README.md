# Denton Chinese Christian Fellowship Website

Welcome to the official website of Denton Chinese Christian Fellowship!

## About This Website

This website serves our church community by providing:
- Service times and weekly schedule
- Information about our beliefs and mission
- Upcoming events and activities
- Contact information

## How to View

Visit our website at: https://denton-ccf.github.io/

## Customization

To customize this website:
1. Edit the `index.html` file
2. Update fellowship name, address, contact info, and schedule
3. Modify colors, images, and content as needed
